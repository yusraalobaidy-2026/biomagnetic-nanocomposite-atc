# biomagnetic-nanocomposite-atc
This repository presents a green-synthesized, biomagnetic CS/Fe3O4 nanocomposite, created using Ficus carica extract, that exhibits high selectivity against aggressive Anaplastic Thyroid Carcinoma (ATC). Computational analysis confirmed robust chemisorption, while it revealed significant ROS-driven cytotoxic effects and high biocompatibility.
