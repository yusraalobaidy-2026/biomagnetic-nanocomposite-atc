# biomagnetic-nanocomposite-atc
 Green Synthesis, Characterization, and Computational Insights of Biomagnetic CS/Fe3O4 Nanocomposite for Selective Cytotoxicity Against Aggressive Anaplastic Thyroid CarcinomaThis repository hosts the computational framework, dataset, and analysis tools for evaluating a novel, green-synthesized biomagnetic CS/Fe₃O₄ nanocomposite targeting aggressive Anaplastic Thyroid Carcinoma (ATC).
 
 📢 Availability Notice⚠️
 Data & Code Release: All datasets, figures, and simulator source code will be fully available and open-sourced immediately after the peer-reviewed paper is officially accepted.
 
 🔬 Overview & Key Findings
 1. Green Synthesis & CharacterizationMethod: Multi-step green synthesis utilizing Ficus carica (Fig) leaf extract for the Fe₃O₄ core reduction.
  
📢  Machine Learning: Machine-learning interatomic potentials (MACE-MP-0).Insights ⚠️: 
 
 Confirmed coordinate covalent bonding and Ligand-to-Metal Charge Transfer (LMCT).Thermodynamics: Highly exothermic adsorption energy, proving strong chemisorption, exceptional thermal stability, and mechanical resilience.
 Biological & Cytotoxic EvaluationTarget: Selective treatment against treatment-resistant Anaplastic Thyroid Carcinoma (ATC).
